# Sales-Data-Analysis
Data analysis project using Python, Pandas, and visualization tools

## Overview
This project analyzes sales data to extract meaningful business insights.

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Insights
- Technology category generates the highest revenue among   
  all categories, despite fewer transactions compared to Office Supplies.         
- The West region contributes the highest overall sales, indicating
  strong market performance in that area. 
- Sales show noticeable growth during the end of the year
  (especially November–December), suggesting seasonal demand patterns.
- Standard Class shipping mode accounts for the highest sales,
  likely due to its cost-effectiveness and frequent usage.
- A few sub-categories dominate total sales, indicating that
  customer demand is concentrated on specific product types.
- Sales are highly concentrated in a few states, with top-performing
  states contributing a major portion of total revenue.
- The Consumer segment generates the highest revenue, making it the
  most valuable customer group.
